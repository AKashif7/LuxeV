User profile

The user profile takes in multiple user inputs, for multiple questions asked via prompts, in an attempt to build a user profile for the user.

The questions are presented in html to the user, and user inputs are taken from promts.

JavaScript is used to take in user inputs for multiple questions by presenting corresponding prompts, and it is used to check for validation. If the user input is valid, the answer to a question on the page is updated to reflect their input. Otherwise questions are marked as unanswered.

JavaScript is also used to load a section of questions at a time, and to allow for question answers to be edited.
